# Experiment 01 - Symbol Table Implementation

## Objective
Develop a lexical analyzer to recognize tokens and create a symbol table while recognizing identifiers.

## Compilation
```bash
gcc symbol_table.c -o symbol_table
```

## Run
```bash
./symbol_table
```

## Sample I/O
```
Enter your choice
1.Create  2.Insert  3.Modify  4.Search  5.Display  6.Exit: 1
Enter number of entries: 2
Enter variable and value:
a 10
b 20
VARIABLE    VALUE
a           10
b           20
```
