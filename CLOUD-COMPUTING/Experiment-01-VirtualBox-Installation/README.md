# Experiment 01 - Virtual Workstation Installation (VirtualBox)

## Objective
Install VirtualBox open-source cloud workstation with Linux/Windows OS on top of Windows 8 and above.

## Steps
1. Download VirtualBox from https://www.virtualbox.org/wiki/Downloads
2. Run installer and follow the wizard
3. Download a Linux ISO (Ubuntu recommended)
4. Open VirtualBox Manager -> **New**
5. Set Name, Type (Linux), Version (Ubuntu 64-bit)
6. Memory: minimum 1024 MB
7. Disk: minimum 10 GB (VDI, Dynamically allocated)
8. Click **Finish** -> Select ISO -> **Start**
9. Follow the Ubuntu installation wizard

## Result
VirtualBox installed and Guest OS (Ubuntu) configured successfully.
